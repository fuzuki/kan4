簡単官報管理官。
=============

web版pdf官報(<https://kanpou.npb.go.jp/>)をダウンロードして、手元で管理するソフトです。

## 機能  
* pdf官報ダウンロード
* pdf連結
* 見出し検索

## 動作環境  
Windows8以降(.NET Framework 4.5 以降)

## 開発環境  
Visual Studio 2015
